App not returning to normal after I press the minimize button (three lines located top right of the window)
- Go to your display settings and check the SCALE and LAOUT
- Set it to 100%
- If it's on 100% already, set it to something else, then apply. Then change it back to 100% and apply
No audio playing?
Old version
Please make sure you are on the latest version. Anything above 1.2.8.8 as any version bellow that will not work anymore

Windows Media Player
A user managed to fix this issue by checking his codecs. See if windows media player can actually play mp3 files

FIX
1. Open CMD, By opening the start menu and type in CMD
2. run this command:
Dism /Online /Enable-Feature /FeatureName:WindowsMediaPlayer /All
Antivirus is detecting the app as a virus
- Please let us know who makes your antivirus as we will need to contact the provider. Contact us via #help--support
- Add https://lukeairtool.net/ to ignore list on your antivirus
- Add AirTool.exe to ignore list on your antivirus
