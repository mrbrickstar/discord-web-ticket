An App to create and manage Web Tickets through Discord

## Note:

This was originally created for LukeAirTool.net. So there might be some names or something that I forgot to replace, I apologize for this


# How to install


1. Download the code by clicking the "Download" Button and extract it
2. Install node.js [here](https://nodejs.org/en/)
3. Install all necessary Packages (express, discord.js, multer, path, fs, node-iplocate, request-ip)
4. Run the script with `node index.js`
5. 
# Configure the Tool
To customize the tool to your brand or server, follow this steps

1. Replace every `localhost` in the code through your domain (e.g. http://localhost/ through https://myserver.com)
2. Replace the Logo and the header of the ticket (which is nice.html(don't ask why I named this nice.html)) through your Brand- or Server name
3. Replace the logo and the heading of the ticket page
